**Recruitment Performance & Placement Analytics Dashboard**

Project Overview:
This project analyzes recruitment performance, recruiter efficiency, hiring funnel effectiveness and placement outcomes using Excel and Power BI.

Tools Used:

Power BI
DAX
Data Cleaning
Data Visualization

KPIs:

Total Submissions
Total Interviews
Total Offers
Total Placements
Submission Ratio
Interview Ratio
Offer Ratio
Placement Ratio
Recruiter Conversion %

**Dashboard Features**

Recruiter Analysis:
Recruiter Performance
Recruiter Efficiency
Conversion Tracking

Client Analysis:
Placements by Client
Client Contribution

Funnel Analysis:
Leads → Submissions → Interviews → Offers → Placements

Trend Analysis:
Monthly Hiring Trends
